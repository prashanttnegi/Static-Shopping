First open the command promt for the code folder.
In the command prompt, give the command : "pip install -r requirement.txt "
Wait for the installation of all the packages.
After installation of packages, give the command: "python app.py"
It will run a server, then open any of the browser.
In browser's url bar, enter: "http://127.0.0.1:5000/"
Now you can access the application.
